📋 Instrucciones Completas - GitHub Pages

🆓 COMPLETAMENTE GRATUITO PARA SIEMPRE

🔸 Paso 1: Crear Cuenta en GitHub

1. Ve a github.com

2. Registra una cuenta gratuita

3. Verifica tu email

🔸 Paso 2: Crear Repositorio

1. Haz clic en el botón "+" → "New repository"

2. Nombre: prestamos-app-v2 (o el que prefieras)

3. ✅ Marcar "Public"

4. ✅ Marcar "Add a README file"

5. Clic en "Create repository"

🔸 Paso 3: Crear Personal Access Token

1. Ve a Settings → Developer settings → Personal access tokens

2. "Generate new token (classic)"

3. Scopes: ✅ repo (completo)

4. Copia el token (ghp_xxxxx)

🔸 Paso 4: Activar GitHub Pages

1. En tu repositorio: Settings → Pages

2. Source: Deploy from a branch

3. Branch: main → Save
