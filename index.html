📋 Instrucciones Completas - GitHub Pages (100% GRATIS)

🆓 COMPLETAMENTE GRATUITO PARA SIEMPRE

🔸 Paso 1: Crear Cuenta en GitHub (si no tienes)

1. Ve a github.com

2. Registra una cuenta gratuita

3. Verifica tu email

🔸 Paso 2: Crear Repositorio

1. Haz clic en el botón "+" → "New repository"

2. Nombre: prestamos-app (o el que prefieras)

3. ✅ Marcar "Public"

4. ✅ Marcar "Add a README file"

5. Clic en "Create repository"

🔸 Paso 3: Crear Personal Access Token

1. Ve a Settings (en tu perfil, esquina superior derecha)

2. Developer settings → Personal access tokens → Tokens (classic)

3. "Generate new token" → "Generate new token (classic)"

4. Note: "Prestamos App"

5. Expiration: No expiration

6. Scopes: ✅ repo (marca toda la sección)

7. "Generate token"

8. ¡IMPORTANTE! Copia el token inmediatamente (empieza con ghp_)

🔸 Paso 4: Subir Archivos al Repositorio

1. En tu repositorio, clic en "Add file" → "Create new file"

2. Nombre del archivo: index.html

3. Copia y pega todo el código HTML de esta aplicación

4. Commit: "Add loan management system"

5. "Commit new file"

🔸 Paso 5: Crear archivo de datos

1. "Add file" → "Create new file"

2. Nombre: prestamos.json

3. Contenido:

[]

4. "Commit new file"

🔸 Paso 6: Activar GitHub Pages

1. En tu repositorio, ve a "Settings"

2. En el menú lateral: "Pages"

3. Source: Deploy from a branch

4. Branch: main

5. Folder: / (root)

6. "Save"

7. ¡En unos minutos tu app estará en: https://tu-usuario.github.io/prestamos-app/

🔸 Paso 7: Configurar la Aplicación

1. Abre tu aplicación en: https://tu-usuario.github.io/prestamos-app/

2. Completa los campos:

  • Usuario GitHub: tu-usuario
  • Repositorio: prestamos-app
  • Token: ghp_xxxxxxxxx

3. "Conectar con GitHub"

4. ¡Listo! 🎉

✨ Beneficios de esta configuración:

  • ✅ 100% Gratuito para siempre
  • ✅ Acceso desde cualquier dispositivo
  • ✅ Sincronización automática
  • ✅ Backup automático en GitHub
  • ✅ URL profesional
  • ✅ Sin límites de uso
🔒 Seguridad: Tu token solo se guarda en tu navegador y nunca se comparte. Es completamente seguro.
